# Data-Science-jobs-analysis-in-Excel
Excel Project: Cleaning and Analysis of Data Science jobs data
## Overview
This project analyzes a dataset of job salaries to extract meaningful insights about industries, roles, and locations.
## Data Cleaning Steps
- Index dropped – Removed unnecessary row indexing.
- Salary column parsed – Converted salary data to numeric format for analysis.
- Description table dropped – Removed verbose textual descriptions that were not useful.
- Useless columns removed – Cleaned the dataset by removing irrelevant columns.
## Analysis Steps
- Pivot table created – Summarized salary data by industry, role, and location.
- Dashboard created – Built visualizations to highlight trends and insights for easier interpretation.
## Key Insights
- Senior and Associate Data Scientists get the highest paying jobs.
- Salaies are highest in Aerospace and Defense industries while low in Accounting and Marketing jobs.
